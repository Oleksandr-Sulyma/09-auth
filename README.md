# NoteHub

Simple and efficient note management application built with **Next.js**, **React Query**, and **TypeScript**.

## Demo

[09-auth-nine-gilt.vercel.app](https://09-auth-nine-gilt.vercel.app)  

## Repository

[GitHub](https://github.com/Oleksandr-Sulyma/09-auth)  

## Features

- User authentication (Sign up / Login / Logout)  
- Profile management (Edit username and avatar)  
- Create, view, edit, and delete notes  
- Filter notes by tags  
- Pagination and search with debounced input  
- Modal preview for notes  
- Server-side data fetching with React Query  
- Responsive layout  

## Tech Stack

- Next.js 14  
- React 18  
- TypeScript  
- React Query / TanStack Query  
- CSS Modules  
- Axios for API requests
